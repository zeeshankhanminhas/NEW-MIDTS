## CTA Issue

### Problem

Generic CTAs reduce clarity and action.

### Fix

Use one primary CTA only:

- Start Your Project

Alternative options:

- Get a Quote in 24 Hours
- Submit Your Design Request

---

## Missing Trust Signals

### Problem

B2B users require confidence before engaging.

Currently missing:

- Proof of capability
- Process clarity
- Credibility markers

### Fix

Add trust statements:

- Confidential handling of all engineering data
- UK-based coordination
- Dedicated engineering support

Future enhancements:

- Sample outputs
- Case studies
- Workflow diagrams

---

## Funnel Not Clearly Defined

### Problem

The backend system is strong, but the frontend does not communicate the process.

### Fix — Add How It Works Section

How it works:

1. Submit your requirement
2. Share project details
3. Receive quote
4. We execute

## Over-Design Risk

### Problem

Excessive motion and animation can reduce clarity.

### Rule

Design supports clarity — not replaces it.

1. **Messaging Clarity** — Fix hero section immediately.

5. **Reduce Visual Noise** — Keep UI minimal and purposeful.
